# HelpDog

Landing page da campanha **HelpDog** — ajude a alimentar cães de rua.

## Stack

- Vue 3 + TypeScript
- Vite
- CSS próprio (layout fiel ao design)

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Seções

1. Hero com CTA **AJUDAR AGORA**
2. Sobre — “Os cães falam…”
3. Produtos — Special Dog, Biscrok e Drools
